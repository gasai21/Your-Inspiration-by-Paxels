# 📱 Your Inspiration by Pexels

**Your Inspiration by Pexels** adalah aplikasi Android untuk mencari dan mengeksplorasi gambar inspiratif menggunakan Pexels API. Aplikasi ini dibangun dengan UI modern berbasis Material Design dan memberikan pengalaman pengguna yang nyaman melalui dukungan tema terang dan gelap.

---

## ✨ Fitur Utama

- 🏠 **Home Screen**  
  Menampilkan gambar populer (curated) dari Pexels.

- 🔍 **Search Screen**  
  Mencari gambar berdasarkan keyword secara real-time.

- 📄 **Detail Screen**  
  Menampilkan detail gambar dengan resolusi tinggi.

- ❤️ **Favorite Screen**  
  Menyimpan gambar favorit menggunakan penyimpanan internal.

- ⚙️ **Settings Screen**  
  Mengatur preferensi aplikasi seperti tema.

---

## 🎨 Tema

Aplikasi mendukung:

- 🌙 Dark Mode  
- ☀️ Light Mode  

Pengguna dapat mengganti tema sesuai preferensi melalui halaman Settings.

---

## 🛠️ Teknologi yang Digunakan

- **Kotlin**
- **Jetpack Compose**
- **Retrofit** (untuk komunikasi API)
- **Local Storage** (untuk menyimpan favorite)
- **Material 3**
